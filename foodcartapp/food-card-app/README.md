# AI Powered Food Delivery Platform

## Features

- Next.js Routing
- Apollo GraphQL
- Firebase Authentication
- Firestore Database
- Redux Toolkit
- Redux Thunk
- Responsive UI
- Order History
- Static Generation
- Real-time Firestore Updates

npm install
npm run dev
